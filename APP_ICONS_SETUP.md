## 🚀 How to Generate All Platform Icons

### Step 1: Export to GitHub & Pull Locally
```bash
# After exporting to GitHub
git clone <your-repo>
cd <your-repo>
npm install
```

### Step 2: Generate All Platform-Specific Assets
```bash
# This command generates all required icon sizes for iOS and Android
npx @capacitor/assets generate --iconBackgroundColor '#f97316' --splashBackgroundColor '#f97316'
```

This will automatically create:
- **iOS Icons**: All required sizes (20x20 to 1024x1024)
- **Android Icons**: Adaptive icons, launcher icons, and notification icons
- **Splash Screens**: iOS and Android launch screens in all required sizes

### Step 3: Verify Generated Assets
After running the command, check these folders:

### Step 4: Sync to Native Projects
```bash
npx cap sync
```

## 🎨 Customizing Your Icons

### Option 1: Use Your Own Icon Image
If you have your own logo:

1. **Replace the app icon**:
   - Save your logo as `public/app-icon.png` (1024x1024px, PNG)
   - Must be square with transparent or solid background
   - Should work well at small sizes

2. **Replace the splash screen**:
   - Save your splash as `public/splash-screen.png` (1080x1920px, PNG)
   - Portrait orientation for mobile
   - Center important content (safe area)

3. **Regenerate**:
   ```bash
   npx @capacitor/assets generate
   ```

### Option 2: Customize Colors
Edit `assets.config.json`:
```json
{
  "iconBackgroundColor": "#YOUR_COLOR",
  "splashBackgroundColor": "#YOUR_COLOR",
  "splashBackgroundColorDark": "#YOUR_DARK_COLOR"
}
```

Then regenerate:
```bash
npx @capacitor/assets generate
```

### Option 3: Advanced Configuration
Create `capacitor.config.json` for more control:
```json
{
  "resources": {
    "icon": {
      "sources": ["public/app-icon.png"],
      "backgroundColor": "#f97316"
    },
    "splash": {
      "sources": ["public/splash-screen.png"],
      "backgroundColor": "#f97316",
      "showSpinner": false
    }
  }
}
```

## 📱 Icon Design Best Practices

### App Icon Guidelines:
- **Size**: 1024x1024px minimum
- **Format**: PNG with transparency or solid background
- **Content**: Simple, recognizable at small sizes
- **Colors**: High contrast, limited color palette
- **Safe Area**: Keep important content in center 80%
- **No Text**: Icons should be primarily visual/logo

### Splash Screen Guidelines:
- **Size**: 1080x1920px (portrait) or 1920x1080px (landscape)
- **Safe Area**: Center content in middle 70% of screen
- **Branding**: Include logo and app name
- **Simple**: Avoid complex details
- **Fast Loading**: Should feel instant (<2 seconds)

### iOS Specific:
- iOS applies automatic rounding to icon corners
- Design with a square canvas, don't pre-round
- Avoid small text or intricate details
- Test on actual devices in different sizes

### Android Specific:
- Android supports adaptive icons (foreground + background)
- Design can be masked in different shapes (circle, square, squircle)
- Keep important content in center "safe zone"
- Consider both light and dark theme appearances

## 📦 App Store Requirements

### iOS App Store:
- **App Icon**: 1024x1024px, PNG (no transparency)
- **Required Sizes**: Automatically generated by Xcode
- **Review**: No text overlays, no screenshots
- **Format**: RGB color space, no alpha channel for store icon

### Google Play Store:
- **App Icon**: 512x512px, PNG (32-bit with alpha)
- **Feature Graphic**: 1024x500px for store listing
- **Screenshots**: Multiple sizes required
- **Content Rating**: Icons must be appropriate for rating

## 🛠️ Troubleshooting

### Icons not updating?
```bash
# Clean build folders
rm -rf ios/App/App/Assets.xcassets
rm -rf android/app/src/main/res/mipmap-*

# Regenerate
npx @capacitor/assets generate
npx cap sync

# Clean native builds
cd ios && pod install && cd ..
cd android && ./gradlew clean && cd ..
```

### Splash screen not showing?
1. Check `capacitor.config.ts` splash screen settings
2. Verify image exists: `public/splash-screen.png`
3. Check image format (PNG, RGB, no CMYK)
4. Rebuild native apps completely

### Wrong colors appearing?
1. Clear cache: `rm -rf node_modules/.cache`
2. Check `assets.config.json` color values
3. Regenerate with explicit colors:
   ```bash
   npx @capacitor/assets generate --splashBackgroundColor '#f97316'
   ```

### Icons look blurry?
1. Ensure source image is high resolution (1024x1024+)
2. Use PNG format (not JPEG)
3. Avoid scaling up low-res images
4. Check if transparency is causing issues
